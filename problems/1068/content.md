# 盛最多水的容器

## 题目描述

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。
返回容器可以储存的最大水量。

说明：你不能倾斜容器。

## 输入格式

第一行一个正整数n,表示有n条线段。

第二行n个正整数height[1],height[2],……，height[2]，意义如题。

## 输出格式

输出一个正整数，表示容器可以储存的最大水量

## 输入样例

```plaintext
9
1 8 6 2 5 4 8 3 7
```

## 输出样例

```plaintext
49
```
## 数据范围与提示

- 对于50%的数据， 2<=n<=1000
- 对于100%的数据，2<=n<=105 , 0<=height[i]<=10^4
